# Claude Code — Starter Guide

A friendly, beginner-focused guide to setting up **Claude Code** as an AI business partner — written for a small retail shop owner on Windows.

**📖 Read it live:** https://acmondano.github.io/claude-code-starter-guide/

Covers: installing on Windows, the Claude Pro plan, giving Claude a memory (`CLAUDE.md`), connectors, skills (including a ready-to-use retail contract maker), real day-to-day workflows, safety, and a printable cheat-sheet.

Single self-contained `index.html` — no build step, works offline if you download it.
